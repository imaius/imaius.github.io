centos7系统更换内核

  • centos
  • 内核

  前面写过centos7配合bbr搭建ssr，效果挺不错的，今天重启了机器，原因未知发现bbr失效了，查询内核发现不是最新的了，在此记录更换内核的方式。

  使用下面的命令查看已经安装的内核：

awk -F\' '$1=="menuentry " {print i++ " : " $2}' /etc/grub2.cfg

我的机子得到如下结果：

  把CentOS Linux (4.11.3-1.el7.elrepo.x86_64) 7 (Core)作为启用内核

grub2-set-default 2

  然后reboot即可。重启后

lsmod | grep bbr

  得到

tcp_bbr                16384  24

  发现bbr开启了。至此恢复正常使用。

  如果你想要删除旧的内核可以如下操作：

查看当前内核

rpm -qa | grep kernel

然后直接yum删除

yum -y remove '内核名'

值得注意的是，如此操作过后需要重新操作一下开始的步骤，因为序号可能变动了。
